======================================================
            Jogo da Memória - Readme.txt
======================================================

Bem-vindo ao Jogo da Memória!

Este é um projeto de jogo da memória desenvolvido como parte de um trabalho de programação. Abaixo estão algumas informações sobre o projeto e como executá-lo.

1. Estrutura de Arquivos:
   - index.html: Página principal com o menu de login e registro.
   - CriarConta.html: Página de criação de conta de usuário.
   - Login.html: Página de login de usuário.
   - Jogo.html: Página do jogo da memória.
   - styles.css: Arquivo CSS para estilos comuns.
   - jogo.css: Arquivo CSS específico para o jogo da memória.
   - script/LoginCriar.js: Script JavaScript para gerenciar o registro de usuário.
   - script/jogo.js: Script JavaScript para a lógica do jogo da memória.

2. Instruções de Uso:
   - Para começar, abra o arquivo index.html em seu navegador.
   - Você pode se registrar como um novo usuário na página CriarConta.html.
   - Após o registro, faça login na página Login.html.
   - Uma vez logado, você será redirecionado para a página do jogo Jogo.html.
   - Clique no botão "Start" para iniciar o jogo.
   - Encontre todos os pares de cartas correspondentes o mais rápido possível.
   - Sua pontuação será baseada no tempo que leva para completar o jogo.
   - Clique no botão "Reset" para reiniciar o jogo a qualquer momento.
   - Para voltar ao menu principal, clique no botão "Menu Principal" durante o jogo.

3. Pontuação:
   - A pontuação do jogador é calculada com base no tempo que leva para completar o jogo.
   - Quanto mais rápido o jogador completar o jogo, maior será a pontuação.
   - A pontuação atual do jogador é exibida na página do jogo.
   - A melhor pontuação é salva no Local Storage e pode ser vista na próxima vez que o jogador jogar.

4. Notas Adicionais:
   - Certifique-se de ter o JavaScript habilitado em seu navegador para o funcionamento adequado do jogo.
   - Este projeto foi desenvolvido como parte de um trabalho de programação e pode ser personalizado e aprimorado conforme necessário.

Divirta-se jogando o Jogo da Memória!
